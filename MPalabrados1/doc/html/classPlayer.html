Descripción detallada

Class used to store the letters owned by the player, which is continuosly changing according to the evolution of the game. It is stored as a CSTRING, with a \0 delimiter and all the internal operations must be carried out EXCLUSIVELY with the functions provided in <cstring> See http://www.cplusplus.com/reference/cstring/ for details about cstrings.

All the operations in this class must leave the set of letters ordered lexicographically

Please note that all the characters are stored in ISO8859 standard and in uppercase**
